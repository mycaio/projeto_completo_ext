# LIGA JÁ Front-end

Protótipo React para a interface do projeto, usando `useEffect`, `useState` e `props`.

## Como usar

1. Abra um terminal em `frontend`
2. Execute:

```bash
npm install
npm run dev
```

3. Abra a URL exibida pelo Vite (por padrão `http://localhost:4173`).

## O que está pronto

- Carga inicial de estado via `useEffect`
- Estado de jogador e missão gerenciado com `useState`
- Componentes reutilizáveis via `props`
- Comandos iniciais prontos para integração com back-end
- Simulação de API em `src/services/api.js`
